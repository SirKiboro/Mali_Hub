# MaliHub

MaliHub is a scalable campus marketplace platform that enables students to buy, sell, and trade goods efficiently. Built with a focus on performance, security, and production-grade backend architecture.

## Features
- User authentication and authorization
- Product listing and search
- Secure transactions
- RESTful API design
- Scalable architecture

## Tech Stack
- Java
- Spring Boot
- PostgreSQL / MySQL
- REST APIs

## Project Status
In active development.
